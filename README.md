- upgrade to Node 9.2.0

# Setup

you can access from http or https (updated)

## Install

yoy can use docker to run it,

# to build:
docker build -t entity .

# run
docker run -p 3000:3000 -d entity 
or
node app.js

# see images
docker images

# see process id
docker ps -a

# stop container
docker stop <CONTAINER_ID>
